Project Name

Thrive 3

Business Context

Thrive 3 is a modern fashion and streetwear e-commerce brand selling clothing and accessories across multiple categories and sizes. The platform must be conversion-focused, scalable, and ready for backend, database, and logistics integrations.

Tech Stack (Preferred)

Framework: Next.js (latest, App Router)

Language: TypeScript

Styling: Tailwind CSS

Animations: Framer Motion

State Management: Context / Zustand / Redux Toolkit (choose best fit)

Backend Readiness: API routes or server actions

Database Ready: Schema-friendly architecture (Prisma-compatible)

Auth: Login / Signup (JWT / session-ready structure)

SEO: Server-side rendering, metadata, structured data
